# AvatarComponent

`AvatarComponent` is a wrapper around PrimeVue `Avatar` that standardizes avatar behavior in WeFa:

- initials fallback based on `username`
- deterministic background color for non-image avatars
- optional explicit `label`
- optional `imageUrl`
- `square` and `circle` shapes

## Basic Usage

```vue
<script setup lang="ts">
import AvatarComponent from '@/components/AvatarComponent/AvatarComponent.vue'
</script>

<template>
  <AvatarComponent username="Ada Lovelace" />
</template>
```
