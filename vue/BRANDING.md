# Branding

## Wording 

### Company name 
The company name should always be written in capital letters and it should never be altered in spacing or spelling.

## Core Colors

### Text

Text should always be set in dark blue, white or
dark gray.

- Hex #535F6B
- Hex #162950
- Hex #66728c
- Hex #A5BEDB
- Hex #E7EBF5

### Primary Colors

The 3 primary colors should first be used in their main version.
Different color tints are also available and can be used with their
parent color to create layered effects in single design.

#### Teal

- Main #00ACC2
- #40C1D1
- #9FE0E8
- #DFF5F7
